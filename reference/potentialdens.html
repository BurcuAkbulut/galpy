galpy.potential.Potential.dens¶

Potential.dens(R, z, phi=0.0, t=0.0, forcepoisson=False)¶

NAME:

dens

PURPOSE:

evaluate the density rho(R,z,t)

INPUT:

R - Cylindrical Galactocentric radius

z - vertical height

phi - azimuth (optional)

t - time (optional)

KEYWORDS:

forcepoisson= if True, calculate the density through the Poisson
equation, even if an explicit expression for the density exists

OUTPUT:

rho (R,z,phi,t)

HISTORY:

2010-08-08 - Written - Bovy (NYU)
