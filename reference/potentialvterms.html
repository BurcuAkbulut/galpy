galpy.potential.vterm(Pot, l, deg=True)¶

NAME:

vterm

PURPOSE:

calculate the terminal velocity at l in this potential

INPUT:

Pot - Potential instance

l - Galactic longitude [deg/rad]

deg= if True (default), l in deg

OUTPUT:

terminal velocity
